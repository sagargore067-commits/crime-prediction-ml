# AI-Powered Crime Prediction System

## Overview
This project predicts crime category based on day, district, and hour using Machine Learning.

## Dataset
Kaggle Crime Dataset

## Technologies
- Python
- Random Forest
- Streamlit

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Train model:
   python main.py

3. Run app:
   streamlit run app/app.py
